Pulsedb

Pulsedb is a time series database server and library.

You can use it for storing metrics like network usage, load, etc..

You can use it as an embedded library for Erlang servers or use it as a standalone HTTP server.

Server features:

  • store data with second precision
  • automatically aggregate to minutes
  • cleanup old data with second precision
  • aggregate and downsample on request
  • subscribe to queries via Websockets
